Chosen Colour map & motivation:

Since 2015, the authors of SciPy have set the matplotlib default color map to Viridis. This color map was designed to improve graph readability for readers with common forms of color blindness and/or color vision deficiency. The color maps are also perceptually-uniform, both in regular form and also when converted to black-and-white for printing. I use seaborn that is build on matplotlib, and thus uses Viridis as well.

Since then, other outlets such as this one have adapted the Viridis standard, which leads me to believe that it should be a suitable, color-blind proof color hue.
